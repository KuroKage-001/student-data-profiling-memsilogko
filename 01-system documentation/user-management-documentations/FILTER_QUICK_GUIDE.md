# User Management Filters - Quick Guide

## Quick Overview
Filter users by role and status to quickly find specific accounts.

## Available Filters

### Role Filter
- All Roles (default)
- Administrator
- Department Chairman
- Faculty
- Student

### Status Filter
- All Statuses (default)
- Active
- Inactive
- Suspended

## How to Use

### Desktop
1. Filters are always visible below the search bar
2. Select role from first dropdown
3. Select status from second dropdown
4. Results update automatically

### Mobile
1. Tap "Filters" button to show/hide
2. Badge shows number of active filters
3. Select filters from dropdowns
4. Results update automatically

## Combining Filters

### With Search
1. Set filters (e.g., Role: Student)
2. Type in search box (e.g., "john")
3. Results show students with "john" in name/email

### Multiple Filters
1. Set Role filter (e.g., Faculty)
2. Set Status filter (e.g., Active)
3. Results show only active faculty

## Clearing Filters

### Clear All
1. Set Role to "All Roles"
2. Set Status to "All Statuses"
3. Clear search box

### Clear One Filter
- Change dropdown back to "All" option
- Other filters remain active

## Filter Count Badge

### Mobile Only
- Shows number next to "Filters" button
- Updates when filters change
- Example: "Filters (2)" means 2 active filters

## Keyboard Shortcuts

- **Tab:** Move between filters
- **Arrow Keys:** Navigate dropdown options
- **Enter:** Select option
- **Escape:** Close dropdown

## Troubleshooting

### No Results Showing
- Check if filters are too restrictive
- Try "All" options
- Clear search box

### Filters Not Working
- Refresh the page
- Check internet connection
- Clear browser cache

### Mobile Toggle Not Working
- Tap the "Filters" button
- Check if filters are already visible
- Try refreshing the page
